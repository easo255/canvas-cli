## Setup

 - Create a file named `config.json` in the src folder that looks like this:
 
 ```` 
{
    "canvasToken": "",
    "canvasUrl": ""
 } 
 ````
 
 And populate it with a token generated from canvas account settings, and url to your canvas LMS.
